답변은 우선 차트 제목과 설명을 번호 머리글 스타일로 간략하게 정리해.
모두 정리한 이후, 그 아래에는 각각의 차트를 통해 어떤 인사이트를 분석 가능한지 추가 설명을 작성할 것.

표는 "차트명 | 차트종류 | 행영역 | 열영역 | 값영역 | 정렬또는필터방식 | 피벗테이블명(영문, 간략히)" 으로 구성할 것.
표 아래에는 각 차트를 통해 어떤 인사이트를 분석 가능한지 추가 설명을 간략히 정리해.

#차트

#VBA코드작성지침
- 대시보드 틀을 제작할 시트의 이름 : "대시보드"
- 대시보드의 전반적인 스타일 : 뉴모피즘, 모던 스타일 대시보드
- 대시보드의 테마색은 다음 색상을 사용할거야. 배경색 : F2EDF3, 제목영역색: 1D2433, 차트영역색: FFFFFF
- 시트의 채우기 색상을 배경색으로 변경.
- 시트 최상단에는 제목영역색으로 채워진 가로로 긴 타이틀 바를 추가하고 안에는 "글로벌 게임 TOP 1000 판매 현황 대시보드"으로 제목을 넣을 것.
- 타이틀 바 아래에는 차트영역색으로 채워진 모서리가 둥근 사각형을 2개 추가할 것.
- 그 아래에는 차트영역색으로 채워진 모서리가 둥근 사각형을 3개 추가할 것.
- 둥근 사각형의 이름은 ChartBox1, ChartBox2, ... 로 순번이 1씩 증가하는 이름으로 설정.
- 각 둥근 사각형에는 제목영역의 색, 85%의 투명도, 30pt 정도의 흐림도, 바깥쪽 방향 45도 각도의 그림자를 추가해.
- VBA 코드를 실행할 때 오류가 발생하지 않도록, 단계별로 차근차근 꼼꼼히 검토해서 작성할 것.

#코드작성지침
- "피벗"이라는 이름을 가진 시트에 피벗테이블을 생성할거야.
- 피벗캐시를 생성할 땐, 테이블을 참조하므로 relativeTo 속성은 비워둬서 오류가 발생하지 않도록 주의해.
- 피벗테이블을 추가할 땐, 각 피벗테이블이 겹치지 않도록 이전 단계에서 생성한 피벗테이블의 마지막 열을 찾은 후, 해당 열의 3열 오른쪽에 새로운 피벗테이블이 추가되도록 할 것.
- 피벗테이블의 구성은 아래 피벗테이블 구성 표를 참고해.
- 피벗테이블의 값필드를 추가할 땐 이름 중복으로 인한 오류가 발생하지 않도록, 레이블 뒤에 합계 또는 평균을 붙여.
- 정렬이 필요할 경우 AutoSort를 사용하고, Top10 항목의 값필터는 PivotFilters.Add2 속성을 사용해.
- VBA 코드 를 실행할 때 오류가 발생하지 않도록, 단계별로 꼼꼼히 검토해서 차근차근 작성할 것.

#VBA코드작성지침
- 대시보드에 사용할 차트는 "대시보드" 시트에 있는 ChartBox1, ChartBox2.. 형식으로 1씩 증가하는 순번의 이름을 가진 도형 위에 추가.
- 아래 차트종류및구성 표의 순서에 따라 각 도형을 먼저 찾은 후, 도형의 위치와 크기를 참조하여 차트를 생성.
- 차트의 높이와 너비는 도형 높이와 너비의 90% 크기로 생성하고, 각 도형의 좌측상단 위치를 기준으로 도형의 높이 및 너비의 5% 여백을 두고 생성.
- 피벗 차트의 버튼은 ShowAllFieldButtons 속성을 false로 설정해서 숨기기.
- 차트의 눈금선과 테두리를 제거하고, 범례(legend)가 1개일 경우 범례도 제거할 것.
- 차트의 배경 채우기 색은 투명하게 설정.
- 차트의 제목은 아래 목록의 차트명으로 수정
- 가로막대 차트일 경우 첫번째 세로축의 ReversePlotOrder 를 True로 설정해서 항목을 거꾸로 표시.
- 코드를 실행할 때 오류가 발생하지 않도록, 단계별로 꼼꼼히 검토해서 차근차근 작성할 것.
